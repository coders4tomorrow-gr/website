Coding with Scratch

  • When: December 29, 2022
  • For: Students grades 4 - 8
  • Where: Kentwood KDL

In this 4-hour program, Coders4Tomorrow will teach the basics of coding using the programming platform Scratch, a beginner-friendly environment where students can create their own video games.
Projects include: a Mario-themed platformer, Flappy Bird, Cookie Clicker, and more! For grades 4-8. No prior coding experience is necessary.

To register for this course, please see KDL's website.
